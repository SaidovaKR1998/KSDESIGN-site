# KS DESIGN - Сайт дизайнера интерьеров

![Python](https://img.shields.io/badge/Python-3.11%2B-blue)
![Django](https://img.shields.io/badge/Django-4.2-green)
![Bootstrap](https://img.shields.io/badge/Bootstrap-5-purple)
![License](https://img.shields.io/badge/License-MIT-yellow)

Сайт-портфолио для дизайнера интерьеров Кристины. Разработан на Django с использованием Bootstrap.

## 📋 О проекте

Сайт представляет собой профессиональное портфолио с возможностью:
- Просмотра проектов с фотогалереей
- Скачивания PDF-презентаций проектов
- Ознакомления с услугами и ценами
- Отправки заявок через форму обратной связи
- Скачивания бриф-опросника для клиентов

## 🚀 Функционал

### Для посетителей:
- **Главная страница** - информация о дизайнере, статистика, призыв к действию
- **Портфолио** - все проекты с фильтрацией по категориям
- **Детальная страница проекта** - фотогалерея, описание, скачивание PDF
- **Услуги** - тарифы и что входит в каждый пакет
- **Этапы работы** - подробное описание процесса
- **Контакты** - форма обратной связи, ссылки на соцсети

### Для администратора (через админку):
- Управление проектами (добавление фото, PDF, описаний)
- Редактирование услуг и цен
- Просмотр заявок от клиентов
- Изменение контента на страницах

## 🛠 Технологии

- **Backend:** Python 3.11+, Django 4.2
- **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript
- **База данных:** SQLite (разработка), PostgreSQL (продакшн)
- **Дополнительно:** django-ckeditor, Pillow, Gunicorn

## 📦 Установка и запуск

### Клонирование репозитория

git clone https://github.com/ваш_логин/ksdesign-site.git
cd ksdesign-site

### Создание виртуального окружения

python -m venv venv
venv\Scripts\activate

### Установка зависимостей

pip install -r requirements.txt

### Настройка базы данных

python manage.py migrate
python manage.py createsuperuser

### Запуск сервера

python manage.py runserver

## 👩‍💻 Для заказчика

Вход в админку:
- Перейдите по адресу: ваш_сайт/admin
- Введите логин и пароль
- Управляйте контентом через интуитивный интерфейс

Как добавить новый проект:
1. Зайдите в админку
2. Выберите "Projects" → "Add"
3. Заполните:
- Название проекта
- Категория
- Описание
- Площадь, год
- Главное фото
- PDF-файл (если есть)
4. Сохраните

Как посмотреть заявки:
1. В админке выберите "Applications"
2. Отметьте обработанные заявки

## 🚀 Деплой на сервер

Подготовка
1. Убедитесь, что DEBUG = False в settings.py
2. Добавьте ваш домен в ALLOWED_HOSTS
3. Настройте базу данных для продакшена
